Steps to set the environment up:
1. pip3 install -r requirements.txt
2. install java rte 1.8, create system variable JAVA_HOME & add path to java to PATH system variable
3. if you work on windows, download below binaries to path c:\hadoop\bin and add that directory to PATH system variable
https://github.com/songdongsheng/hadoop-native/tree/master/3.1.1/Windows-x64
4. set HADOOP_HOME system variable, pointing to c:\hadoop
